1. WHAT IS MACHINE LEARNING (ML)?
======================================================================
Traditional Programming relies on a software engineer writing strict rules.
Machine Learning flips this: you provide Data and Answers, and the computer
finds the underlying rules/patterns.

Traditional: Data + Rules  ---------> [ Computer ] ---------> Answers
ML:          Data + Answers -------> [ Computer ] ---------> Rules/Model


======================================================================
2. THE CORE PARADIGMS OF LEARNING
======================================================================

A. Supervised Learning
   - The model learns from "labeled" data (input-output pairs).
   - Goal: Predict the output for new, unseen inputs.
   - Subtypes:
     * Regression: Predicting continuous numeric values (e.g., house prices).
     * Classification: Predicting discrete categories (e.g., Spam vs. Not Spam).

B. Unsupervised Learning
   - The model works with "unlabeled" data.
   - Goal: Discover hidden patterns or structures within the data.
   - Subtypes:
     * Clustering: Grouping similar data points together (e.g., customer segments).
     * Dimensionality Reduction: Compressing data while keeping key info.

C. Semi-Supervised Learning
   - Uses a small amount of labeled data combined with a large amount of 
     unlabeled data. Useful when data labeling is expensive.

D. Reinforcement Learning (RL)
   - An Agent learns by interacting with an Environment.
   - It receives Rewards for good actions and Punishments for bad ones.
   - Goal: Maximize cumulative reward over time (used in gaming, robotics).

3. ESSENTIAL MATHEMATICAL PILLARS
======================================================================
You do not need to be a mathematician, but ML algorithms rely heavily on:

- Linear Algebra: Vectors, Matrices, and Matrix Multiplication (how computers 
  store and process large data features quickly).
- Calculus: Derivatives and Gradients (used to tweak and optimize models 
  during training via "Gradient Descent").
- Probability & Statistics: Mean, variance, distributions, and Bayes' Theorem 
  (used to deal with uncertainty and make predictions).


======================================================================
4. DATA PREPROCESSING & FEATURE ENGINEERING
======================================================================
Garbage In = Garbage Out. Raw data is rarely ready for a machine learning model.

Key Concepts:
- Missing Value Imputation: Filling blank data spots with the mean, median, 
  or removing them entirely.
- Handling Categorical Data: Converting text (like "Red", "Blue") into numbers 
  using techniques like One-Hot Encoding or Label Encoding.
- Feature Scaling: Normalizing or Standardizing numbers so features with huge 
  ranges (like Salary) don't dominate features with small ranges (like Age).
- Feature Selection: Dropping irrelevant or redundant data columns to simplify 
  the model and speed up training.

5. CORE SUPERVISED LEARNING ALGORITHMS
======================================================================

- Linear Regression:
  Fits a straight line (y = mx + c) to find relationships between numeric variables.

- Logistic Regression:
  Despite the name, this is used for Classification. It outputs a probability 
  between 0 and 1 using a Sigmoid curve.

- Decision Trees:
  Asks a series of If-Else questions to split data down into branches until 
  it reaches a final prediction.

- Random Forest:
  An "Ensemble" method. It builds dozens of different Decision Trees and 
  takes their majority vote to make a highly accurate prediction.

- Support Vector Machines (SVM):
  Finds the optimal boundary line (hyperplane) that maximizes the margin/space 
  between different classes of data.


======================================================================
6. CORE UNSUPERVISED LEARNING ALGORITHMS
======================================================================

- K-Means Clustering:
  Randomly places "K" center points (centroids), groups data points closest 
  to them, shifts the centers, and repeats until data is split into K clusters.

- Principal Component Analysis (PCA):
  A dimensionality reduction technique that squashes high-dimensional data 
  (e.g., a table with 100 columns) down to 2 or 3 columns while retaining 
  most of the variance/information.

7. MODEL EVALUATION & VALIDATION
======================================================================
Never evaluate your model on the same data it used to learn!

Data Splitting Strategy:
[---------------------------- Total Data ----------------------------]
[      Train Set (70-80%)      |  Validation Set (10%)  | Test Set (10%) ]
  (Used to learn patterns)       (To tune parameters)    (Final Exam)

Metrics for Classification:
- Accuracy: Correct predictions divided by total predictions.
- Precision: Out of all predicted positives, how many were actually positive?
- Recall: Out of all actual positives, how many did the model find?
- F1-Score: The balance (harmonic mean) between Precision and Recall.

Metrics for Regression:
- MAE (Mean Absolute Error): Average absolute difference between predicted 
  and actual values.
- RMSE (Root Mean Squared Error): Penalizes larger errors more heavily.

8. OVERFITTING, UNDERFITTING, & REGULARIZATION
======================================================================

- Underfitting (High Bias):
  The model is too simple to learn the data pattern (e.g., using a straight 
  line for complex curved data). Performs poorly on both train and test data.

- Overfitting (High Variance):
  The model memorizes the training data perfectly, including its noise and 
  random errors. It performs amazingly on training data, but fails completely 
  on new, unseen test data.

- Regularization (L1 / L2):
  Techniques (like Lasso or Ridge) that add a penalty to complex models to 
  prevent them from overfitting, keeping the model simpler and more stable.


======================================================================
9. INTRODUCTION TO DEEP LEARNING
======================================================================
Deep Learning is a specialized sub-field of ML inspired by the human brain.

- Neural Networks: Layers of interconnected nodes (neurons).
  * Input Layer: Receives the data features.
  * Hidden Layers: Extract increasingly complex patterns.
  * Output Layer: Produces the final prediction.

- Popular Architectures:
  * CNNs (Convolutional Neural Networks): Great for computer vision/images.
  * RNNs / Transformers: Great for sequential data like text and language.
